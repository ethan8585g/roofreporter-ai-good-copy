# Backlink Outreach — Free Tools Embed Seeding

Generated as part of Phase 8 SEO work. The goal: seed the 5 embeddable free tools
at `/tools/pitch-calculator`, `/tools/material-estimator`, `/tools/shingle-calculator`,
`/tools/insurance-deductible-estimator`, and `/tools/solar-production-estimator` across
~50 roofing directories, trade blogs, supplier sites, and insurance resource pages.

Every embed carries a do-follow credit link back to `www.roofmanager.ca` via the
snippet shown on each tool page (Copy snippet button).

## How to use this list

1. Grab the embed snippet from the tool page (`/tools/<tool>` → "Copy snippet").
2. Email or DM the contact with a short pitch (template at the bottom of this file).
3. Track outreach in `leads` table under `source = 'backlink-outreach'` so the team
   can follow up on placements.
4. For each placement that goes live, verify the backlink is do-follow, then log
   in `/admin/seo/backlinks` (if built) or a Google Sheet.
