# Alberta / Edmonton Local-Authority Backlink Checklist

Manual outreach deliverable for **Quick Win #3** in [BACKLINK_STRATEGY_2026.md](BACKLINK_STRATEGY_2026.md). All items below produce permanent local-pack signals from high-trust .ca domains. None of this requires code — it's a person submitting profiles or contributing content.

**Owner:** _assign before starting_
**Target completion:** within 14 days

> **Note on street address.** The current public NAP only has "Alberta, CA". Several association directories (especially ECA) require a full street address. Decide between (a) registering a real or virtual business address, or (b) submitting the limited NAP and accepting some directories will reject. Resolve before starting.

## Tracking

For each submission, log to the `backlinks` table once it's built (QW6). Required fields:

- `target_domain`, `target_url`, `anchor_text`, `dofollow`, `placement_date`
- `outreach_status`: `submitted` → `live` → `verified-dofollow`
- `notes`: any required maintenance (e.g., ACA renews annually)

Until the table exists, log to a Google Sheet and back-fill on QW6 completion.

## Why these matter (one-liners)

- **ECA, ACA, COAA, ICBA**: high-trust .ca construction-vertical authorities → directly improves Edmonton/Alberta local pack rank
- **Startup Edmonton, Alberta Innovates, Edmonton Unlimited**: gov-adjacent tech directories → diversifies link profile beyond construction vertical, signals "tech company" classification
- **Edmonton Journal, CTV/Global**: news mentions are the strongest E-E-A-T signal possible for a regional brand
- **University of Alberta, NAIT**: .edu-equivalent links are rare and valuable; even a single one significantly lifts Domain Rating
